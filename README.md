# Ligeirinho Bebidas

Site institucional da **Ligeirinho Bebidas** — marca de bebidas refrescantes e energéticas com foco em leveza, sabor e energia para o dia a dia.

## Estrutura

```
├── index.html          # Página de entrada (splash)
├── home.html           # Página inicial
├── servicos.html       # Catálogo de produtos
├── empresa.html        # Sobre a marca
├── contato.html        # Contato e distribuição
├── components/
│   ├── navbar.html
│   └── footer.html
├── components.js       # Carregamento de navbar e footer
└── img/                # Imagens e ícones
```

## Como visualizar localmente

```bash
python3 -m http.server 8080
```

Abra [http://localhost:8080](http://localhost:8080) no navegador.

## GitHub Pages

Publique a branch `main` (ou `gh-pages`) com a raiz do repositório como source. O `index.html` na raiz é a landing page.

## Tecnologias

- HTML5
- Tailwind CSS (CDN)
- Google Fonts (Inter, Manrope, Noto Serif)
- Material Symbols
